# Cahier des charges — CuisineEnsemble

## 1. Contexte

CuisineEnsemble est une plateforme de repas partagés qui permet aux habitants d’un quartier de proposer ou rejoindre des repas communautaires.

## 2. Problématique

Comment faciliter l’organisation de repas partagés tout en prenant en compte les contraintes alimentaires, la confiance entre utilisateurs et la coordination du groupe ?

## 3. Objectifs

- Réduire l’isolement alimentaire
- Favoriser les rencontres locales
- Découvrir de nouvelles cuisines
- Partager les coûts des repas
- Sécuriser l’information sur les allergies et régimes alimentaires

## 4. Acteurs

### Hôte / cuisinier
- Créer un repas
- Fixer prix, places, lieu et date
- Renseigner les allergies
- Accepter ou refuser des participants
- Discuter avec le groupe
- Recevoir une note

### Convive
- Voir les repas disponibles
- Filtrer selon prix, quartier, régime
- Réserver une place
- Signaler ses allergies
- Participer au chat
- Noter le cuisinier

### Administrateur
- Modérer les repas
- Gérer les utilisateurs
- Traiter les signalements
- Suivre les statistiques

## 5. MVP

- Authentification
- Profil utilisateur
- Gestion des repas
- Réservation
- Allergies et régimes
- Chat de groupe
- Notation
- Signalement simple
- Dashboard admin
