## Cartes MVP prioritaires

### Authentification
- Créer la page inscription
- Créer la page connexion
- Sécuriser les mots de passe
- Générer un token JWT
- Protéger les pages privées

### Profil utilisateur
- Afficher les informations utilisateur
- Modifier quartier
- Modifier allergies
- Modifier préférences alimentaires

### Hôte
- Créer la page “Proposer un repas”
- Enregistrer le repas en base
- Afficher “Mes repas proposés”
- Annuler un repas
- Voir les demandes de réservation
- Accepter/refuser des participants

### Convive
- Afficher la liste des repas
- Ajouter filtres prix/quartier/régime
- Voir le détail d’un repas
- Réserver une place
- Voir “Mes réservations”
- Annuler une réservation

### Chat
- Créer un chat par repas
- Envoyer un message
- Afficher les messages
- Limiter l’accès au chat aux participants acceptés

### Avis
- Noter le cuisinier
- Noter un convive
- Calculer la note moyenne

### Admin
- Dashboard admin
- Gestion des utilisateurs
- Gestion des repas
- Gestion des signalements
- Statistiques générales
