# Étapes GitHub

1. Créer un nouveau repository GitHub nommé `CuisineEnsemble`.
2. Dans le dossier du projet :

```bash
git init
git add .
git commit -m "Initial commit - CuisineEnsemble"
git branch -M main
git remote add origin https://github.com/TON-UTILISATEUR/CuisineEnsemble.git
git push -u origin main
```

3. Pour cloner ensuite :

```bash
git clone https://github.com/TON-UTILISATEUR/CuisineEnsemble.git
```
